# Miraziz Admin

Простая веб-админка для просмотра и управления заявками. Без сборки — чистый HTML/CSS/JS.

## Возможности
- Список всех заявок с сортировкой по дате
- Фильтр по статусу (Новые / В работе / Готово / Отклонено)
- Поиск по имени, телефону, e-mail, сообщению
- Создание, редактирование, удаление заявок
- Быстрая смена статуса прямо в таблице
- Индикатор состояния API и БД

## Запуск
Сначала запустите бэкенд (см. `../backend/README.md`), затем откройте админку.

Проще всего — любым статическим сервером:
```bash
cd admin
python3 -m http.server 5500
# затем откройте http://localhost:5500
```
Или просто откройте `index.html` в браузере (файл `file://` тоже работает, но статический сервер надёжнее из-за CORS).

## Настройка адреса API
Если бэкенд не на `http://localhost:4000`, поменяйте константу `API` в начале файла `app.js`.
